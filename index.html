Litestomp

The C library used to talk the Stomp which is a simple to implement client protocol for working with ActiveMQ and other messaging systems.

Introduction:

This is a fork of the old libstomp library, modified for simplicity and reusability. This library implements a simple STOMP, a text oriented messaging protocol that can be used to talk to message brokers such as Apace ActiveMQ.

This library supports the version 1.2 of the STOMP protocol as documented by: https://stomp.github.io/stomp-specification-1.2.html.

Project documentation, as well as examples, can be found here: http://orpiske.net/files/litestomp/html/index.html.

Please feel free to contribute to make this library better.
